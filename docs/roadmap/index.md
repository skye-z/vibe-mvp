# 路线图概览

VibeMVP 采用 MVP（最小可行产品）方法进行项目管理开发。

## 当前状态

| 版本 | 状态 | 描述 |
|------|------|------|
| v1.0 | 已发布 | 核心插件功能 |
| v1.5 | 计划中 | 模板优化与开发体验改进 |
| v2.0 | 计划中 | 更多模板类型与集成 |
| v3.0 | 远期规划 | 生态系统与高级功能 |

## 开发理念

1. **增量交付** - 尽早交付可用功能
2. **用户反馈** - 根据实际使用情况迭代
3. **质量优先** - 始终保持代码质量

## 版本亮点

### v1.5 - 质量与打磨
专注于模板优化、错误处理和多语言支持。

### v2.0 - 协作
更多模板类型、更好的文档集成。

### v3.0 - 企业级
高级 AI 功能、插件市场和私有化部署选项。
