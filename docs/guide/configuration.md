# 配置

VibeMVP 在 `.mvp/` 目录中使用模板文件进行配置。

## 模板结构

```
.mvp/
├── guide/           # 项目指南
│   ├── index.md     # 主文档
│   └── *.md         # 其他指南文件
├── modules/         # 功能模块
│   └── *.md         # 模块文档
├── roadmap/         # 版本路线图
│   └── v0-*.md      # 版本文件（版本号中的点替换为横杠）
└── zh/              # 中文版本
    ├── guide/
    ├── modules/
    └── roadmap/
```

## 模板变量

模板中使用 `++` 包裹的变量会在初始化时被替换：

| 变量 | 描述 |
|------|------|
| `++PROJECT_NAME++` | 项目名称 |
| `++PROJECT_REQUIREMENTS++` | 项目需求 |

## 路线图版本命名

版本文件命名格式：`v{大版本}-{小版本}.md`

例如：`v0-1.md`、`v1-0.md`
