# 介绍

VibeMVP 是一个面向 OpenCode 的 MVP 风格项目管理与文档生成插件。它帮助开发者系统性地管理项目需求、跟踪开发进度，并自动生成项目文档。

## 功能特性

- **项目初始化** - 使用自然语言初始化项目需求，生成完整的 MVP 模板
- **需求管理** - 添加和管理项目需求，自动更新路线图
- **任务执行** - 按路线图逐步执行开发任务
- **自动文档** - 生成 VitePress 文档，支持中英文双语

## 工作原理

1. **初始化** - 使用 `mvp_init` 命令，用自然语言描述项目需求
2. **生成模板** - 创建 `.mvp/` 目录，包含 guide、modules、roadmap 三个模块
3. **规划版本** - 将需求拆分为至少四个 MVP 版本（初始版、最小可用版、测试版、未来迭代版）
4. **执行任务** - 使用 `mvp_next` 按路线图逐步执行
5. **迭代更新** - 使用 `mvp_add` 添加新需求

## 项目结构

```
.mvp/
├── guide/           # 指南（流程、设定、配置项、特殊要求）
├── modules/         # 模块（功能拆分为不同模块）
├── roadmap/         # 路线图（MVP 版本规划）
└── zh/              # 中文版本
    ├── guide/
    ├── modules/
    └── roadmap/
```
